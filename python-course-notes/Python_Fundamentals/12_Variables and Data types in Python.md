# Variables in Python 
- Data/values can be stored in temporary storage spaces called variables.

# DataTypes In Python
- Every variable is associated with a data-type
## Types of DataTypes in Python:-
- Numbers
- String 
- List
- Tuple
- Dictionary
- Set
- Boolean - TRUE , FALSE
- Float- 3.14, 5.55
- int - 10, 500
- complex - 10 + 5i etc.
